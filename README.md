# space-territory
Another space game (very different from the first)

## To Build game.js

# 1. Setup Browserify
```
npm i -g browserify
```

# 2. Install required modules
```
npm i pixi.js pixi-viewport pixi-keyboard
```

# 3. Build the work-game.js file a browser-functioning file called ./game/game.js
```
build.bat
-- or --
build.sh
```
